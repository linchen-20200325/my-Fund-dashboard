## 專案定位
- **產品**：境外共同基金（保險型保單）戰情室 — TWD 換匯後績效 + 總經訊號 + 危機回測 + AI 策略建議
- **技術棧**：Streamlit + pandas + plotly/altair + FRED/Yahoo/FundClear + Google Sheets + Gemini API + NAS Squid Proxy
- **核心禁令**：🚫 全面排除 ETF / 個股，本系統專注共同基金
- **入口**：`app.py`（Streamlit Cloud 部署）
